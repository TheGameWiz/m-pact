Treat context as a limited working surface. Prefer narrow searches, specific file reads, summarized noisy output, and task-focused checkpoints over broad exploratory dumps.

Avoid kitchen-sink sessions where unrelated tasks, failed approaches, stale logs, and repeated corrections accumulate around the current task.

**How to apply:** Before reading or pasting large material, ask what evidence is actually needed. After repeated failed corrections or unrelated task switches, checkpoint useful findings and restart from cleaner context.
